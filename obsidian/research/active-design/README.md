# Active Design

This directory contains the active design specifications for kalin-wm.

## Structure

Documents are organized in **Obsidian-style** - small, focused files that link together.

## Quick Start

1. Start at [[index]] for decisions summary
2. Follow links to specific topics
3. Cross-references use `[[filename]]` syntax

## Design Decisions

| Aspect | Decision | Document |
|--------|----------|----------|
| Layout | Hybrid (columns + 2D) | [[layout-paradigm]] |
| Navigation | Super+Arrows focus, Super+Shift+Arrows pan | [[navigation]] |
| Placement | Horizontal strip (Niri-style) | [[window-placement]] |
| Overview | Deferred | [[overview-mode]] |
